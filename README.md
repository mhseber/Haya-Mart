🌙 Haya Mart - Premium Modest Islamic Fashion Platform
Haya Mart is a sophisticated e-commerce solution dedicated to promoting and providing high-quality modest Islamic fashion. Built with a focus on elegance and modern web standards, the platform offers a seamless shopping experience through a high-performance Next.js 16 architecture and a stunning Glassmorphic UI.

✨ Core Features
🕋 Modest Collection Showcase: A curated display of Islamic outfits designed with a premium aesthetic.

🎨 Glassmorphic Dark UI: A modern, minimalist interface utilizing Tailwind CSS 4 and DaisyUI for a luxury feel.

⚡ High-Speed Performance: Optimized with Next.js Turbopack for near-instant page transitions and load times.

🛒 Advanced Cart Management: A robust system allowing users to easily manage their selected items.

❤️ Personalized Wishlist: Integrated functionality for users to save their favorite modest styles for later.

🔐 Secure Authentication: User accounts and profiles secured via Next-Auth and Firebase.

📊 Interactive Data: Dynamic dashboards featuring ApexCharts and Recharts for order and sales tracking.

🎭 Fluid Motion: Engaging user interactions powered by Framer Motion and Lottie animations.

🛠️ Technical Architecture
Frontend & Core
Framework: Next.js 16.1.1 (App Router)

Library: React 19

Styling: Tailwind CSS 4.1.13 & DaisyUI 5.1.25

Icons: Lucide-react & React Icons

Backend & Data
Databases: MongoDB & Firebase

Authentication: Next-Auth & Firebase Hooks

State Management: React Hooks (Client-side persistence via LocalStorage)
